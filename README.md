## Getting Started

### Prerequisites
- Go 1.16 or later

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/hieuleminh03/go_practice
   ```
2. Navigate to the project directory:
   ```bash
   cd go_practice
   ```
3. Install dependencies:
   ```bash
   go mod tidy
   ```
